How to use MathJax and xy-package in html

The following steps will enable you to display mathematical symbols and diagrams (written in LaTeX) in your html web page.
  1. Download latest version of MathJax from [here]

  2. Extract the "MathJax-master.zip" into your working directory, where the html file is located. It is better to place it to top directory, e.g., in "public_html", where your html files are located.

  3. Download XyJax from [here], and extract it.

  4. There is a file "xypic.js" inside the extracted folder "XyJax-master/extensions/TeX/". Copy the file "xypic.js" and paste it into "MathJax-master/extensions/TeX/".

  5. Make sure that the permissions level of the folder "MathJax-master/"(including all files and subfolders inside it) is appropriate to be accessible for others (only "Access files/subfolders" for "Others" is sufficient).

  6. Open your html file, say "index.html" using any text editor you like [e.g. emacs / gedit / xed / pluma / terminal (e.g., pico index.html)]

  7. Copy the following codes

    8. <script type='text/x-mathjax-config'> MathJax.Hub.Config({ extensions: ['tex2jax.js'], jax: ['input/TeX','output/HTML-CSS'], TeX: {extensions: ['xypic.js','autoload-all.js'], TagSide: 'left'}, tex2jax: {inlineMath: [["DOLLAR","DOLLAR"],["\\(","\\)"]]}, 'HTML-CSS': { scale: 85 } }); </script> <script type="text/javascript" src="MathJax-master/MathJax.js"> </script> and paste it inside <head> ... </head> of your html file.
    *** Please replace 'DOLLAR' with '$' symbol in the above code.

  8. Now you can write any mathematical expressions within dollar or double dollar symbols, and MathJax will automatically generate the math symbols in your html page. To draw commutative diagrams, please use "\xymatrix{...}" package.

    Warning! Please note that "tikz" packages are NOT supported yet.

  9. Examples of diagrams using "\xymatrix" package (Please write the codes within double dollar symbols as in LaTeX) :

    Example 1. Code: \xymatrix{ \mathcal{F}(X) \ar[r]^{T_X} \ar[d]_{\mathcal{F}(\varphi)} & \mathcal{G}(X) \ar[d]^{\mathcal{G}(\varphi)} \\ \mathcal{F}(Y) \ar[r]^{T_Y} & \mathcal{G}(Y) } Output:
    \[ \xymatrix{ \mathcal{F}(X) \ar[r]^{T_X} \ar[d]_{\mathcal{F}(\varphi)} & \mathcal{G}(X) \ar[d]^{\mathcal{G}(\varphi)} \\ \mathcal{F}(Y) \ar[r]^{T_Y} & \mathcal{G}(Y) } \]

    Example 2. Code: \xymatrix{ U \ar@/_/[ddr]_y \ar@{.>}[dr]|{\langle x,y \rangle} \ar@/^/[drr]^x \\ & X \times_Z Y \ar[d]^q \ar[r]_p & X \ar[d]_f \\ & Y \ar[r]^g & Z } Output:
    \[ \xymatrix{ U \ar@/_/[ddr]_y \ar@{.>}[dr]|{\langle x,y \rangle} \ar@/^/[drr]^x \\ & X \times_Z Y \ar[d]^q \ar[r]_p & X \ar[d]_f \\ & Y \ar[r]^g & Z } \]
